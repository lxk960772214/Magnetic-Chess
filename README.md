# 磁力棋

一款双人策略棋盘游戏，玩家轮流放置带有磁力的棋子，利用磁力吸引引发连锁反应。

## 玩法规则

1. 两位玩家轮流在圆形棋盘上放置磁铁棋子
2. 先选择棋子类型，再点击棋盘放置
3. 若放置后引发棋子互相吸引移动，所有位移棋子回到当前玩家手中
4. 吸引可引发连锁反应（雪崩效应）
5. 若未引发位移，则换对方下棋
6. 先放完手中所有棋子者获胜

## 棋子属性

| 棋子 | 质量 | 磁力 | 范围 | 特点 |
|------|------|------|------|------|
| ▲ 三角形 | 小 | 弱 | 小 | 易被拉走，但灵活好放 |
| ● 圆形 | 中 | 中 | 中 | 平衡型 |
| ■ 正方形 | 大 | 强 | 大 | 不易被拉走，但易引发连锁 |

## 如何开始

- **桌面端**：打开 `computer.html`
- **手机端**：打开 `mobile.html`

无需服务器，直接在浏览器中运行即可。

## 文件说明

- `computer.html` — 桌面版（鼠标操作）
- `mobile.html` — 手机版（触控操作，自适应屏幕）
- `落子.mp3` — 落子音效
- `碰撞.mp3` — 磁铁碰撞音效
